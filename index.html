Mutual Fund Portfolio Tracker

This page provides instructions and a code template for a simple mutual fund analysis and tracking system.

Overview

The Python script below combines three main functionalities:

  1. Report Generation: Filters and recommends top mutual funds based on performance metrics.
  2. Portfolio Booking: Creates a virtual portfolio based on the recommendations and a specified investment amount.
  3. Performance Tracking: Calculates and displays the current value and profit/loss of the virtual portfolio.

Prerequisites

You'll need Python, the pandas library, and a database file named mf.db with tables for fund_metrics and nav_history.

How to Use

  1. Run the Analysis: Before running, make sure your database file mf.db is populated.
  2. Book Portfolio: Run python mf_manager.py. The script will first generate the report and then book the portfolio.
  3. Track Performance: To check the portfolio's performance later, run the script again. It will automatically track the existing portfolio.

Deploying on GitHub Pages

GitHub Pages is for static websites (HTML, CSS, JavaScript). You cannot run Python scripts directly on it. To "deploy" this project on GitHub Pages, you would:

  1. Create a GitHub repository.
  2. Save this HTML content as an index.html file in the root of your repository.
  3. Go to your repository's **Settings > Pages** and select your `main` or `master` branch as the source for GitHub Pages.
  4. This will create a public URL (e.g., your-username.github.io/your-repo) where this static webpage will be visible. It serves as a documentation page for your Python script, not a live application.
